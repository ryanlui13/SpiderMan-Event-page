🕸️ What is this?
This is an interactive landing page I built to practice managing user events and dynamic content without using a heavy framework. I wanted to see if I could create a "Single Page App" feel using just vanilla JavaScript.

🚀 The Cool Parts
No Refreshing: I used event listeners and DOM manipulation so that when you click around or submit a form, the page updates instantly without a slow browser reload.

Modular Code: Instead of one giant, messy script, I broke the JavaScript down into smaller functions. It makes it way easier to debug and looks a lot more like how professional React components are organized.

Playable Media: I integrated video content and custom UI elements that stay responsive, whether you're looking at them on a phone or a desktop.

Interactive Forms: There’s a working input section that reacts to what the user types in real-time.

🛠️ Built With
HTML5 & CSS3 (Flexbox for the layout)

JavaScript (ES6) for all the logic and interactivity

Custom Animations to give it that "Spiderman" energy

🧠 What I Learned
The biggest challenge was keeping the UI in sync with the data. It taught me a lot about how to handle "state" manually—making sure that if a user changes something in one section, the rest of the page knows about it immediately.
